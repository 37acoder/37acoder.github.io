# 为什么我要继续写博客

2021年的9月，我重新搭建了我的新的博客，正如你现在浏览的这样。这篇文章想告诉读者，为什么我要重拾写博客的习惯，以及我对我的博客的期望。也算是我的博客的第一篇序。

# 回顾老博客

把新的博客搭建起来之后，又想到回到老博客去看一眼。看了下最后更新的时间，2020年的年初......

从头到尾把自己的博客翻了一遍，发现了几个问题

# 1. 博客的内容界限不明确

有时候上一篇还在讲面试，下一篇就在讲Java，甚至下下篇就开始讲小型项目代码组织方式了。很难向大家表述自己的博客的主要界限。

所以新的博客努力的做好分类，生活和技术要分开，Golang和Python要分开，编程语言和软件架构要分开。尽可能贯彻金字塔原理。

# 2. 技术博客的缺乏连贯性或深度

工作的一年里看的博客的数量其实要比工作前任何一段时间看的博客都要多，看的多了，就知道自己想要什么了，就更容易在一堆信息中发现自己想要的信息。

我一贯主张，博客不是写给自己看的，而是写给大家看的。所以博客的内容是否能够吸引他人，是博客生命力延续的保障。

连贯性可以让读者有更多的粘性，也可以对自己的更新有正反馈。有深度的内容可以让读者对博客内容的期望提升，读者希望能够获取更多有价值、有意义的知识。

新的博客，我讲希望将自己的内容尽可能做到极致，我将会用项目的方式来组织一些连贯的文章，同时单独的文章也会尽可能的产出深度的内容，避免产生噪音。

# 3. 页面设计上

  1. 不简洁

    现代审美极力的追求简洁之道，典型的简洁设计可以从Apple、Material等设计风格体现。那种千禧年代门户网站的网页设计在现在看来完全无法接受。

    不简洁带来的另一个影响就是，无法突出页面的重心，无法做到专注。

  2. 没有风格

    虽然博客的吸引力主要来源于内容的质量，但是页面设计上的第一也是很重要的。许多博主在页面设计上都带有自身显著特征的页面元素，这一点我也希望新的博客可以慢慢找到属于自己的风格。

所以依照上面的这几点，我采用了VuePress作为博客的框架。页面的设计风格尽可能像Vue官方文档一样简洁，并且Vue也给了我自己灵活的修改空间。

# 你们和我为什么不写博客了？

在回顾老博客的同时，也顺便看了眼挂在老博客友链的其他博主。除了Y同学以外，大部分人的博文最后更新时间都停留在了2020年上半年。不过欣慰的是，你们的域名起码还在续费，不像我关站半年多了才想起来。

仔细想一想，你们和我，为什么开始不写博客了？

# 1. 成长是浮躁的生活

计算机科班的学生，大部分都会在大学生活里，搭建一个自己的博客，有一个博客会被认为是一个很酷的事情。当一个人有了博客，大家就开始使用博客进行社交，将各自的博客放到各自的友链里，你少了就好像你融入不了他们的圈子一样。

当然，我做博客或者个人主页，是在初中的时候开始的，说起来貌似是一件很光荣的事，但结果你也能看到，到这个博客刚搭建起来时，也没能做出成果。

在初中的时候，我就经常寻找那种免费建站的网站，域名、主机都是服务商免费提供，我做的事情就是只管配置就好了。

高中毕业后，开始使用虚拟主机和WordPress来搭建个人博客，也申请了自己的第一个域名。说实话，除了虚拟主机对于学生的我有些贵以外，这套方案还是很好的。域名无需备案、无需面对命令行、网页编辑博文发布，体验上还是很流畅的。

上大学后，接触到了GitHub Pages和MarkDown，加上周围同学耳闻目染，最终还是上了静态网站这一条道。对于以文字为主要载体的，交互简单的个人博客来说，这一定是禅。

上面的历程说明，我不是在折腾就是在折腾的路上，追求技术潮流、追求酷炫，却从来没有追求过内容的质量。这也是我要引出的第二点。

# 2. 需要群众的眼睛

正经人谁写日记啊？下贱！

博客不是我的日志，日记是写给自己看的，博客是写给大家看的。有人看了，才有动力去更新博客，才有动力去提高博客的质量。如果没人看，那更新或不更新，又有什么区别呢？我需要从别人的肯定中肯定自己。

这就进入了一种恶性循环，在主体不做出改变的情况下，博主只能越来越不愿意写、博文质量越来越差、读者越来越少。

但这其实是一种典型的正反馈机制，只需要你的临门一脚，只需要一个执着的心，就能打破这种恶性循环，建立正反馈机制。

怎么掳来这些雪亮的眼睛的？

  1. 要勇敢的把自己推上舞台：上台前要比上台后还紧张，所以需要一个幕后推手，这个推手没谁了，只能自己。当犹豫的时候，就不要犹豫，先站出来展示自己，即使最终的影响是负面的，那就是我之前抉择的时候手抖了。在关键性内容产出后，我会尽可能的在各个渠道进行推广，包括工作群、朋友圈、掘金等。
  2. 要接受批评，然后不在乎批评：人无完人，我写的博客不是学术论文，一定存在不严谨，一定存在错误。我会感谢所有勘误的读者。因为相信自己能够改变，所以不会在乎你们的批评，请放马过来。博客将会支持评论系统，理性的讨论将会被保留。
